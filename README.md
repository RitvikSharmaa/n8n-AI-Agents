# n8n AI Automation Agents

This repository contains my collection of Production-Grade AI Agents built inside **n8n**.

These agents solve multiple real-world automation cases such as Interview Preparation, Resume Screening, PDF Summarization, RAG Document Retrieval, Order Automation and MCP tool execution.

Each workflow is exportable, reusable, and immediately deployable inside any n8n instance.

## Tech Stack Used

* **n8n (AI Workflows)**
* Google Gemini (LLM)
* Pinecone Vector DB (RAG Agent)
* MongoDB (User Session / Chat Memory)
* Google Sheets (Order Management / Screening Results)
* Cohere Embeddings (RAG Agent)
* PDF → Text Extraction Pipelines
